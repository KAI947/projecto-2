ONE FOR ALL

One For All permite al usuario acceder a la energía almacenada, lo que le otorga al usuario una fuerza sobrehumana mientras esté activo.[1] One For All recibe su nombre del hecho de que puede ser "heredado" por otros. Como una antorcha, se transmite de un usuario a otro. El método de herencia y transferencia de One For All se realiza cuando el destinatario recibe el ADN del usuario anterior, tal como se muestra con All Might, que le da a Izuku su mechón de pelo para éste ingerirlo, lo que permitió a Izuku heredar el One For All. El Don no se transferirá a un nuevo destinatario a menos que el usuario actual lo desee, por lo que One For All no puede ser robado a la fuerza. Por ejemplo, el Asesino de Héroes: Stain ingirió la sangre de Izuku pero no heredó One for All porque él no lo deseaba. Sin embargo, esto no significa que One For All no se pueda transmitir por la fuerza.Además, One For All parece alterar la apariencia del usuario de diferentes maneras. Para Toshinori Yagi, aumenta significativamente sus músculos y masa corporal en general para que se vea como una persona completamente diferente. Para Izuku Midoriya, el uso de One For All hace que la parte potenciada de su cuerpo brille, con líneas en forma de venas que se cruzan entre sí. Sin embargo, se desconoce el aspecto de One For All para sus usuarios anteriores.

Una vez que One For All se transfiere al destinatario, el usuario anterior de One For All todavía puede usar las "brasas sobrantes" del mismo y, por lo tanto, todavía puede acceder al One For All. Sin embargo, estas brasas sobrantes no durarán mucho tiempo y se agotarán lentamente hasta que el anterior portador lo pierda completamente. Según All For One, el uso continuo de estas últimas brasas aumentará rápidamente la pérdida de las mismas. Después de que All Might derrotó a All For One, las ascuas de One For All se apagaron y ahora ni siquiera puede mantener su forma muscular por más de unos segundos. Para poder usar el One For All requiere un entrenamiento intenso y riguroso del cuerpo para evitar que el sucesor se desgarre a causa del poder bruto. Incluso con el entrenamiento, el Don sigue ejerciendo una intensa tensión en el cuerpo del usuario (como cuando Izuku lo utilizó por primera vez (se rompió el brazo derecho y las dos piernas). Sin embargo, no todos los usuarios se destacan igual y algunos pueden acostumbrarse a él con bastante rapidez, como lo señaló Gran Torino, quien fue directamente al entrenamiento de combate real para All Might, ya que este último no tardó mucho en adaptarse a sus nuevos poderes, dominándolo casi al momento.
